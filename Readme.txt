# Welcome to the HDL Learning Repository!

This collection focuses on Hardware Description Languages (HDLs), primarily Verilog, to help new students understand and master HDLs and FPGAs. Each example and project in this repository has been tested on the DE1 SOC board, ensuring reliability and practical application. Explore various Verilog codes, from basic to advanced levels, designed to enhance your knowledge and skills in digital design and FPGA development.
